# Operations Department

**Mission:** Keep the organization running cleanly. No task sits idle. No agent finishes work without updating the record.

**Status:** Permanent — never goes dormant.

## Agents

| Agent | File | Role |
|-------|------|------|
| Chief of Staff | `custom/chief-of-staff.md` | Coordination, routing, command center, Ryan interface |
| Project Manager | `custom/project-manager.md` | Task tracking, queue health, status updates |

## Output directory

`00-command-center/` — NEXT_ACTIONS.md, KNOWLEDGE_BASE.md, OPERATING_PRINCIPLES.md

## Responsibilities

- NEXT_ACTIONS.md is never more than 24 hours stale
- No task sits in QUEUED status more than 48 hours without a status update
- All agent outputs are routed to the correct department and committed to the repo
- Ryan's touchpoints are clearly flagged and never buried in agent output
