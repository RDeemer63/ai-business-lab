# Lessons

The most valuable asset in this organization is not code or clients. It is what the organization has learned.

Every experiment result, decision record, post-mortem, and pattern report lives here. The Librarian, Historian, Pattern Analyst, and Knowledge Curator all write to this directory.

## What lives here

| Folder | Owner | Contents |
|--------|-------|----------|
| decisions/ | Historian | Decision records: what, why, alternatives rejected, outcome |
| experiments/ | Librarian | Validation test results with success metrics and signals |
| post-mortems/ | Historian | Post-mortems for killed businesses and failed experiments |
| pattern-reports/ | Pattern Analyst | Monthly: what's working, what keeps failing, what to change |
| prompt-quality/ | Knowledge Curator | Agent definition change log and quality notes |

## The rule

If it happened, it lives here. Nothing disappears into a conversation thread.
