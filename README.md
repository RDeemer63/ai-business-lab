# AI Business Lab

**Mission:** Build a $5k–$10k/month AI-assisted business with under 5 hours/week of human input.

**Operating model:** Ryan is the board. AI runs the company. The repo is the office.

---

## How this works

Claude Code builds. Codex reviews. ChatGPT strategizes. Ryan approves.

Every agent leaves a trail. Every decision is committed. Every experiment is documented.
